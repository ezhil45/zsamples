Scala v2.10.1 Integration

Description

This samples demonstrates Scala integration through a simple Hello World Web application deployed to Liberty.

This open source integration demonstrates Scala integration through a simple Hello World sample Web application deployed to Liberty.

Instructions

  1. Install the packaged server by typing 'java -jar ScalaSample.jar' and following the instructions.
  2. Start the ScalaSample server by running the command 'bin/server run ScalaSample' from the root of the Liberty profile installation, and confirm the application started by looking in the console output": 
    CWWKZ0001I: Application HelloScalaWeb started in XX.XX seconds.
  3. In your browser, enter the URL for the application: http://localhost:@com.ibm.ws.samples.scala_sample.ScalaSample.http_port@/HelloScalaWeb (where port @com.ibm.ws.samples.scala_sample.ScalaSample.http_port@ assumes the httpEndpoint provided in the sample server.xml has not been modified).
  4. In your browser, you should see a phone book shown up.

Notes

This sample does not include the open source dependencies. The installer will provide a list of the open source pre-requisite libraries and where they can be downloaded from. To help obtaining these libraries, the installer provides the option to download them on your behalf.
